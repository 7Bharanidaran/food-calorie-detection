
# 🍽️ Food Calorie Detection

An AI-based system that detects food items from images and estimates their calorie content using deep learning.

## 📌 Overview
This project uses a deep learning model to:
- Detect food items from images
- Estimate the calorie content of detected food
- Provide nutritional insights based on detected items

## 🛠️ Technologies Used
- Python
- YOLOv8
- OpenCV
- Deep Learning
